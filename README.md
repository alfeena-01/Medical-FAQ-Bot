# 🩺 Medical FAQ Bot

A simple and elegant **Medical FAQ Assistant** built with Python, Streamlit, and Hugging Face Transformers.  
This bot answers common medical questions from a curated FAQ dataset and provides a clean, modern UI for interaction.

---

## ✨ Features
- 📖 Answers medical FAQs from a CSV dataset
- 🎨 Beautiful Streamlit UI with custom styling
- ⚡ Fast and lightweight — runs locally in a virtual environment
- 🩺 Domain-specific responses (medical questions only)
- 🔍 Easy to expand with new FAQs (just update the CSV file)

---

## 🛠️ Tech Stack
- **Python 3.10+**
- **Streamlit** for UI
- **Pandas** for CSV handling
- **Transformers (Hugging Face)** for Q&A pipeline

---
